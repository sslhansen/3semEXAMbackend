Links to endpoints:

/api/dog/add-dog Post method for adding dog, only usable by users
/api/dog/get-dogs/{userName} Gets dogs for a user, only usable by users
/api/dog/get-dogs/breeds Gets all breeds, fetched from link provided
/api/dog-breed/{breedName} Gets information for a breed, fetched from links provided

Admin:

/api/admin/get-calls Gets all breed-calls done, only usable by admin
/api/admin/get-calls/{breedName} Gets all breed-calls done to a specific breed, only usable by admin
